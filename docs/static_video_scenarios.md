# 低成本视频生成方案：静态绘本与有声漫画

本文档分析了通过关闭 AI 视频动画生成（Image-to-Video），仅使用“静态图片 + 旁白 + 运镜”来制作低成本高质量视频的业务场景与优势。

## 1. 成本与效率分析

| 方案                      | 核心成本 (每分钟视频)            | 生成速度        | 适用场景                       |
| :------------------------ | :------------------------------- | :-------------- | :----------------------------- |
| **全动画 (I2V)**          | 高 (Luma/Runway 约 $0.5-$1/次)   | 慢 (需等待渲染) | 电影预告片、动作场面、炫技视频 |
| **静态绘本 (Image Only)** | **极低** (仅 DALL-E 约 $0.04/图) | **极快**        | 故事绘本、知识解说、冥想视频   |

**结论**：去掉动画生成环节，成本可降低 **90%** 以上，且生成速度大幅提升，适合批量化内容生产。

## 2. 推荐业务场景

这种模式下的视频产品形态类似 **“有声绘本”** 或 **“动态 PPT”**，在以下赛道极具竞争力：

### 🧸 儿童绘本 / 睡前故事 (Kids Storybooks)

- **痛点**：家长需要大量、不重复的睡前故事；孩子更关注画面色彩和故事情节，而非复杂的动态。
- **优势**：AI 可无限生成新故事。
- **推荐风格**：Q 版 3D (Pop Mart style)、皮克斯风格 (Pixar style)、水彩插画。
- **变现**：YouTube Kids 频道、抖音/小红书“每日故事”账号、付费音频故事集。

### 📖 有声漫画 (Motion Comics)

- **痛点**：传统漫画阅读累眼，视频化能降低阅读门槛。
- **优势**：通过简单的镜头推拉（Ken Burns Effect）让漫画分镜“动”起来，营造沉浸感。
- **推荐风格**：日漫风格 (Anime)、美漫风格、线稿上色。
- **内容方向**：悬疑推理、微恐怖、民间传说、情感故事。

### 📜 历史与人文解说 (Documentary Style)

- **痛点**：历史题材缺乏真实影像资料。
- **优势**：使用 AI 生成的高质量“油画”或“水墨画”作为空镜头，配合深度解说，质感极佳。
- **推荐风格**：中国水墨画 (Ink Wash Painting)、古典油画、写实摄影。

### 🧘 助眠与冥想 (Meditation & ASMR)

- **痛点**：用户需要解压，画面不能太花哨以免干扰。
- **优势**：几张唯美的治愈系风景图，配合长段的白噪音或引导语。
- **推荐风格**：极简主义、自然风景、梦幻光影。

## 3. 系统配置指南

要在本项目中启用此模式，仅需修改 `config.yaml`：

```yaml
# config.yaml

features:
  # 关闭动画生成，系统将自动跳过 Luma/Stability 环节
  # 直接将 Image 拼接为视频
  enable_animation: false

models:
  # 配合场景选择画风（也可用 category_defaults + styles 更精细地按类目映射）
  # image_style: "Pop Mart style, 3D Chibi character, cute rendering" # 儿童绘本
  image_style: "Traditional Chinese Ink Wash Painting" # 历史/人文解说
```

## 4. 进阶建议 (Roadmap)

为了弥补没有动画的视觉单调，建议优先开启/使用以下能力：

- **Ken Burns 效果**：已内置在视频合成中（对静态图自动做轻微推拉运镜）。
- **转场特效**：可按类目在 `config.yaml` 配置（例如 `crossfade`）。
- **字幕（拼音）**：在 `config.yaml` 开启 `features.enable_subtitles` 或使用 CLI `--subtitles`。
