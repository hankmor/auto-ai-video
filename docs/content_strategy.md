# 内容选题建议（与当前项目一致）

“图片 + 语音/旁白”模式（静态绘本 / 动态 PPT）的核心优势在于**低成本**、**高听觉叙事权重**与**强氛围感**。以下是更适合本项目流水线的内容方向建议。

## 1. 故事类（核心赛道）

- **儿童绘本/寓言故事**
  - **特点**：受众对画面动态要求低，偏好风格统一的插画。
  - **优势**：非常适合批量生产与连载。
- **民间传说/神话志怪**
  - **特点**：需要氛围感与宏大场景（天宫、地府、妖魔）。
  - **优势**：AI 静态图比实拍更容易做出“超现实史诗感”。
- **悬疑/恐怖/刑侦故事**
  - **特点**：恐怖在于“脑补”。
  - **优势**：阴森静态图 + 沉浸式音效（可选）往往更有效。

## 2. 知识科普类（易变现）

- **历史解说/人物传记**
  - **痛点**：缺乏真实影像资料。
  - **优势**：用写实/油画/水墨风格的静态图填补画面。
- **少儿百科/冷知识**
  - **痛点**：核心是信息传递。
  - **优势**：用“关键物体/场景图 + 旁白 + 字幕”即可形成高质量内容。

## 3. 情感/治愈类（高完播）

- **睡前故事**
  - **特点**：节奏慢、氛围柔和。
  - **优势**：静态图 + 低语速 TTS + BGM 很容易形成稳定产品形态。
- **助眠/冥想**
  - **特点**：画面需要稳定，不干扰听觉。
  - **优势**：少量高质量静态图配合长旁白即可。

## 4. 读书分享（连载友好）

- **每日一本书 / 章节解读**
  - **特点**：观众注意力在“听觉 + 字幕”。
  - **优势**：
    - 可将书拆成多集连载（本项目支持 `标题:副标题` 的主题格式）
    - 视觉可以更抽象（概念可视化）而非逐字还原情节

## 🚫 不推荐题材

- **搞笑段子**：依赖肢体语言/表情动态，静态图难以承载笑点。
- **动作/打斗片**：静态画面缺乏连贯性与打击感。

# 适合“图片+语音”（绘本/PPT 式）视频模式的题材分析

“图片 + 语音/旁白”模式（Static Image + Narration）的核心优势在于**低成本**、**高听觉叙事权重**和**强氛围感**。

以下是几类特别适合该模式的题材：

## 1. 故事类 (Storytelling) - 核心赛道

_最匹配当前 auto_maker 项目方向_

- **儿童绘本/寓言故事**
  - **特点**：受众对画面动态要求低，偏好色彩鲜艳、风格统一的静态插画。
  - **优势**：AI 绘图（Flux/Jimeng）极擅长生成此类风格，易于量产。
- **民间传说/神话志怪**
  - **特点**：需要极强的“氛围感”和宏大场景（天宫、地府、妖魔）。
  - **优势**：实拍特效成本极高，而 AI 静态图可以轻松渲染出超现实的史诗感。
- **悬疑/恐怖/刑侦故事**
  - **特点**：恐怖在于“脑补”。
  - **优势**：阴森的静态图 + 沉浸式音效（心跳、风声）比廉价的五毛特效视频更具恐怖谷效应，效果极佳。

## 2. 知识科普类 (Edutainment)

- **历史解说/人物传记**
  - **特点**：缺乏真实的视频影像资料。
  - **优势**：利用 AI 生成“复古油画风”或“老照片风”静态图，完美填补画面空缺。
- **冷知识/百科**
  - **特点**：强调信息传递，画面仅需展示关键物体。
  - **优势**：如“深海生物图鉴”，只需一张高清怪鱼图配合解说即可，无需动态。

## 3. 情感/治愈类 (Emotional/Healing)

- **情感语录/散文诗歌**
  - **特点**：核心是文案与 BGM，画面只需唯美不违和。
  - **优势**：AI 易于批量生产唯美风景、夕阳、背影等“壁纸级”素材。
- **助眠/冥想 (ASMR Visuals)**
  - **特点**：画面需要极度稳定，不干扰听觉。
  - **优势**：单张高质量静态图或微动图循环播放即可。

## 4. 商业/网文推广 (Promotion)

- **网文推文 (小说可视化)**
  - **特点**：更新速度快，情节夸张（爽文）。
  - **优势**：利用高张力静态图快速呈现关键剧情（如“龙王赘婿”），是唯一能跟上网文更新速度的视频形式。

---

## 5. 知识/读书博主 (Daily Book)

_非常适合，是目前的高价值赛道_

- **每日一本书 / 读书分享**
  - **特点**：核心是“干货输出”和“思维升维”。观众的注意力在听觉（解说）和字幕上。
  - **优势**：
    - **视觉抽象化**：不需要还原书中的具体场景，而是将**概念可视化**。例如讲《原子习惯》，生成一张“小雪球滚成大雪球”的扁平插画；讲《三体》，生成一张科幻概念图。
    - **图文匹配度宽容**：甚至可以使用仅仅是美观的背景图（书桌、咖啡、阅读灯），配合金句文字展示。
    - **量产容易**：一本书拆成 3-5 个视频，或每天推一本，素材库（书皮、读书场景、抽象插画）可复用。
    - **支持连载**：使用 `Title:Chapter` 格式（如 `小狗钱钱:第一章`），AI 可精准生成分章节内容，适合做连续剧式解读。
  - **推荐风格**：`flat_tech` (商业/工具书)、`zen` (心理/疗愈)、`watercolor` (文学/小说)。

---

## 🚫 不推荐题材

- **搞笑段子**：过度依赖肢体语言和微表情的动态，静态图很难 get 到笑点。
- **动作/打斗片**：静态图缺乏打击感和连贯性，容易像简陋的动态漫画。
